# 스타세이비어 구원자 가이드 DB v3

## 데이터 출처

- 구원자 이름, 속성, 이미지, 클래스: `https://star-savior-arcana-db.pages.dev/savior`
- PVE 장비 세팅, 여정 아르카나 조합, 특잠, 비고: 사용자가 제공한 Google Sheets  
  `https://docs.google.com/spreadsheets/u/0/d/1rAFFXvhMZTsa736JF_X1vylhpCTw-y_KmEKnmwP5m3w/htmlview`

## 반영 범위

- 구원자 49명
- 구원자별 이미지, 속성, 클래스
- 시트에 등록된 구원자의 PVE 장비 세트
- 시트에 등록된 여정 아르카나 조합
- 특잠 코드와 캐릭터별 비고
- 공통 기준: 목걸이 속도, 별도 언급이 없으면 반지 공격력%
- 시트 공통 안내: 효과 적중 최대 170~180 적정선

제공된 시트는 PVE 전용이므로 PVP 세팅은 임의 생성하지 않고 `해당 시트에 없음`으로 표시합니다.
시트에 없는 구원자 역시 세팅을 추측하지 않고 `원본 자료 미등록`으로 표시합니다.
